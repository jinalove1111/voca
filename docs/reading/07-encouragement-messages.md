# 07 — 격려 문구 500 (Encouragement Messages Asset)

작성일: 2026-07-23 · 작성: docs-maintainer · 상태: **자산(asset) — 코드 미적용**

이 문서는 문장 학습(Sentence Learning)·데일리 미션 등 향후 격려 엔진이
소비할 **한국어 격려 문구 500개**의 원본 자산입니다. 이 문서 자체는
어떤 코드도 변경하지 않으며, 기존 코드의 문구 풀
(`src/utils/paulReactions.js`의 `MESSAGE_POOLS`,
`src/utils/sentenceLearning.js`의 `ENCOURAGE`,
`src/components/SentenceLearningFlow.jsx`의 `STAGE_PRAISE`)도 그대로
둡니다 — 실제 적용은 별도 구현 세션(implementer)의 몫입니다.

## 톤 가이드 (실제 코드에서 실측한 원칙)

- **폴(영국 신사 선생님) 캐릭터**: 과장 없이 따뜻하게, "~해요"체 중심.
  기존 문구("잘했어요!", "괜찮아요, 정답을 확인해봐요",
  paulMemory의 "폴은 다 기억하고 있어요!" 등)와 같은 결의 목소리.
- **징벌/압박/죄책감 언어 0** — `paulMemory.js`의 명시 원칙("왜 안 왔어"
  류 문장 금지)과 `paulReactions.js`의 "오답이지만 절대 혼내지 않음"
  원칙을 그대로 따랐습니다. 마감 압박("빨리", "시간 없어요"),
  상실 위협("~이 사라져요"), 비교·질책 표현을 전부 배제했습니다.
- **반복 없음**: 500개 전부 서로 다른 문장입니다.
- 대상: 초등~중등 학생에게 자연스러운 한국어.

## 카테고리 구성

| 카테고리 | 용도 | 개수 |
|---|---|---|
| A. 정답 | 정답 직후 즉시 피드백 | 120 |
| B. 재도전 | 오답 직후 재시도 격려(질책 없음) | 100 |
| C. 거의 다 왔어 | 근접 오답·목표 근접 상황 | 80 |
| D. 훌륭해 | 큰 성취(연속 정답/마스터/극복) 칭찬 | 100 |
| E. 미션 완료 | 세션/미션 완료 순간 | 100 |
| 합계 | | **500** |

## 소비 시 주의사항 (향후 구현 세션용)

- 이 문구들은 **자산일 뿐 코드에 아직 연결되지 않았습니다.** 실제 적용
  시에는 `paulReactions.js`의 `pickNoRepeat`(연속 반복 방지) 또는
  `sentenceLearning.js`의 결정론 선택(`encouragementFor` — 시드/횟수 기반)
  관례 중 소비처에 맞는 쪽을 따르세요. Sentence Learning 엔진 쪽은
  **무작위 함수 금지**(하네스가 코드 레벨로 단언)이므로 결정론 선택만
  가능합니다.
- 채점이 대소문자/구두점 무관(`normalizeAnswer`)이므로, "대문자를
  확인해봐요" 류의 문구는 의도적으로 넣지 않았습니다.
- C(거의 다 왔어) 카테고리는 "근접 오답"과 "목표 근접(진행률)" 두 상황을
  모두 포함합니다 — 소비 시 상황별로 나눠 쓰려면 근접 오답용(2, 4, 8,
  12, 21, 22, 26, 31, 35, 41, 46, 52, 61, 66, 73, 79번)과 진행률용을
  분리해서 매핑하면 됩니다.
